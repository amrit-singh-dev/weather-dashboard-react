# Weather Dashboard (React)

A modern Mac-style weather dashboard built using React.

## Features
- Dynamic weather background
- °C / °F toggle
- Glassmorphism UI
- 3-day forecast
- LocalStorage support
- Loading animation

## Setup
1. Clone repository
2. Run npm install
3. Create .env file:
   REACT_APP_WEATHER_KEY=your_api_key
4. Run npm start
